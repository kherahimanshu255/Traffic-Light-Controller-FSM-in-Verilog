# Traffic-Light-Controller-FSM-in-Verilog
<p>This project demonstrates the design of a Moore Finite State Machine (FSM) to manage a two-way intersection.</p>
<p><b>Design Specifications</b></p>
FSM Type: 4-state Moore Machine.<br><br>
States: <br>
S0: North-South Green / East-West Red.<br>
S1: North-South Yellow / East-West Red.<br>
S2: North-South Red / East-West Green.<br>
S3: North-South Red / East-West Yellow<br><br>
